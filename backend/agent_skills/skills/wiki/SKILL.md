# Навык: Вики

Личная база знаний — markdown-файлы в рабочей директории.

- Перед ответом ищи по файлам (`Grep`, `Glob`) и читай нужные страницы (`Read`). Не
  выдумывай того, чего нет в вики — если не нашёл, так и скажи.
- Создавай и правь страницы по просьбе: `Edit` для точечных правок, `Write` для новых
  файлов. Пиши чистый, аккуратный markdown.
- Держи структуру: осмысленные имена файлов и папок, заголовки, связи между страницами
  относительными ссылками вида `[текст](путь.md)`.
- Помогай с организацией (переименовать, разбить, объединить страницы) аккуратно. Для
  крупной реорганизации можно делегировать субагенту `librarian` через `Task`.

Если запрос на самом деле про дела/напоминания/планы — переключись на навык `tasks`.
